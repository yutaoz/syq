# syq
A command line system info tool for Windows


## Usage
Download the latest release here: https://github.com/yutaoz/syq/releases

In cmd/powershell/your terminal, run ```syq``` in the folder that syq.exe is installed.

Or, add the folder the ```path/to/syq/``` folder to the PATH environment variable to call syq from anywhere.


## Build
Requires CMake and Ninja

Clone the repo, and create a build folder in root

Run ```cmake -G Ninja ..``` in the build folder

Run ```ninja``` to build
